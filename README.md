# Twinkle Days
Syshack2025初陣の春　🏆Sysken賞
## プロジェクト概要
"継続"を助ける記録アプリ<br/>
目標を達成すると星をgetし、夜空に輝かせる⭐️
## 使用技術
フロントエンド：React
バックエンド：Python(FastAPI)
データベース：SQLite3
## セットアップ
1. リポジトリをクローン<br/>
`git clone git`<br/>
`cd frontend`
2. バックエンドで仮想環境を構築<br/>
cd backend
python -m venv venv
アプリの立ち上げ<br/>
フロントエンド<br/>
`cd frontend`
`npm install`
`npm run dev`
